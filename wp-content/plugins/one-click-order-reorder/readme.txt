Place any previous WooCommerce orders again into cart without any restrictions of orders status by just ONE CLICK.

== Description ==

It is used to add any previous order products to cart.
After installing this plugin a button having name **Re-Order** will be added at Order listing section 
on **My Account** page next to each orders as well as on order detail page.

= Main features =
* **Simplest way to place the WooCommerce's previous orders again**
* One click re-order our previous orders.
* If product is not avaliable then notify users.
* Adds previous order products again to the cart.
* **No restrictions of order status. Order with any status can be added again to cart**.
* **Orders may be place from order's listing( My Account) page or order's detail page as well**.
* **Empty cart** after adding the items of an order to cart.
* **Place Same Order** named button added to place the same order by managing the products and direct redirect to checkout.
* **Now the plugin contains basket feature, to add bulk of products into basket and then add them in just one click.**
* **User can save their basket items and can order them anytime.**
* Exclude products when placing the same order again.
* manage quantity of products when placing the same order again.
* Saves user's precious time by adding multiple products of an order at once.
* Easy customization.
* Easy to use and install.
* Easy to configure.

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of One Click Order Re-Order , log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "One Click Order Re-Order" and click Search Plugins. Once you've found our One Click Order Re-Order plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading our One Click Order Re-Order plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

== Frequently Asked Questions ==

= WooCommerce already provides Order Again Feature, So Why we use this plugin ? =

WooCommerce Provides Order Again feature only for completed orders, But Through this plugin you can Re-order your all the orders either the order is completed
or not.
